所有用户提出的需求都必须通过团队流程执行，不能跳过直接开发。

**Why:** 用户明确要求 "以后我提的任何需求都要让团队执行"、"每个需求也都要执行"，之前直接开发没有走流程被指出。

**How to apply:**
1. 收到需求后，自动执行完整团队流程直到代码提交完成：@architect 设计 → @coder 实现 → @tester 审查 → @coder 修复建议项 → 提交代码 → 向用户汇报最终结果
2. 即使是小改动也要走团队流程
3. 每个需求完成后，更新 `project_overview.md` 中的项目进展和待开发计划
